# Events

**Route:** `/events`

## Purpose
The master calendar and activity feed for all in-game events — both recurring and one-time — across the galaxy.

## Key Features
- Calendar view (daily, weekly, monthly) of scheduled events
- Event categories: combat, trade, exploration, community, holiday
- Event detail: duration, location, rules, rewards
- Personal event participation history
- Notification sign-up per event type
- Admin-created special event announcements
