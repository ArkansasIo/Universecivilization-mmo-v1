# Galactic Calendar

**Route:** `/galactic-calendar`

## Purpose
An in-universe calendar system tracking time across the galaxy — including celestial cycles, faction holidays, and historical epochs.

## Key Features
- Custom calendar with galactic date and time
- Celestial event predictions (eclipses, comet passes, supernovae)
- Faction holiday and observances listings
- Historical timeline of major galactic events
- Personal event marking and reminders
- Timezone conversion to real-world time
