# Achievements

**Route:** `/achievements`

## Purpose
Tracks player achievements — permanent badges of accomplishment across all gameplay categories, with rewards and prestige.

## Key Features
- Achievement categories: combat, trade, exploration, social, events
- Progress tracker per achievement (hidden and visible)
- Rewards: titles, cosmetics, credits, stat boosts
- Rare and secret achievements with discovery conditions
- Global achievement rarity percentages
- Showcase display for profile
