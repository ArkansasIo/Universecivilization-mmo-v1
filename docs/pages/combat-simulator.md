# Combat Simulator

**Route:** `/combat-simulator`

**Purpose:** Sandbox battle simulation for testing fleet and ground compositions without resource cost.

**Key Features:**
- Free-form fleet/unit composition on both sides
- No resource expenditure — purely for theorycrafting
- Run unlimited simulations and compare results

**UI Sections:**
- **Attacker Loadout** — configure attacker ships, units, officers, and formation
- **Defender Loadout** — configure defender ships, units, defenses, and formation
- **Simulate Button** — run the battle simulation
- **Results Panel** — detailed outcome report with stats, casualties, and probability analysis
- **History** — log of previous simulations for comparison
