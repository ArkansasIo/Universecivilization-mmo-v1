# Pirates

**Route:** `/pirates`

## Purpose
Tracks pirate factions, their activity hotspots, and player bounties against them — serving as the hub for anti-piracy operations.

## Key Features
- Known pirate factions with territories and strength ratings
- Pirate activity heatmap across systems
- Pirate-hunting missions and fleet deployment
- Player reputation with pirate factions
- Pirate vessel catalog (types, armaments, loot tables)
- Alliance anti-piracy contributions and rewards
