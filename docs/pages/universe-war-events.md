# Universe War Events

**Route:** `/universe-war-events`

## Purpose
Large-scale, server-wide war events that span multiple galaxies — faction vs. faction conflicts with universe-altering stakes.

## Key Features
- Active war event overview with participating factions
- Multi-front battle tracking and sector control percentages
- Event phases: mobilization, conflict, resolution
- Global contribution leaderboard per faction
- Exclusive war bonds and victory rewards
- Lore updates and narrative progression as the war evolves
