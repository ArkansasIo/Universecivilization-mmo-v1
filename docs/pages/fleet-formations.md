# Fleet Formations

**Route:** `/fleet-formations`

**Purpose:** Configure fleet formation layouts for combat bonuses.

**Key Features:**
- Uses `useFleetFormations` for formation state and switching
- Multiple formation types (e.g., Vanguard, Shield Wall, Flanking)
- Each formation grants different stat bonuses to ships in specific positions

**UI Sections:**
- **Formation Selector** — radio/card picker for formation type
- **Formation Grid** — visual layout showing ship positions in the chosen formation
- **Stat Bonuses** — per-position and per-formation stat modifiers displayed on hover/select
