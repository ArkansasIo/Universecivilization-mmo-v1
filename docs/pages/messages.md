# Messages

**Route:** `/messages`

## Purpose
The in-game mail system for direct communication between players, factions, and NPC entities — supporting text, attachments, and structured diplomatic proposals.

## Key Features
- Inbox, sent, drafts, and archived folders
- Compose with formatting and attach items/credits
- Read receipts and message delivery status
- Conversation threading
- Filter and search by sender, subject, date
- Block and report spam functionality
