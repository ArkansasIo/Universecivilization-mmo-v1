# Seasonal Events

**Route:** `/seasonal-events`

## Purpose
Time-limited seasonal events tied to real-world holidays or in-game cosmic phenomena — offering exclusive rewards and themed content.

## Key Features
- Current and upcoming seasonal event listings
- Event progression track with milestone rewards
- Exclusive seasonal currency and loot
- Themed cosmetics, ships, and titles
- Community goal tracking (global participation milestones)
- Archive of past seasonal events and rewards
