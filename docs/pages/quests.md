# Quests

**Route:** `/quests`

## Purpose
The quest journal — tracking active, available, and completed missions from NPC factions, storylines, and procedural generators.

## Key Features
- Quest log with active, available, and completed categories
- Quest detail view: objectives, rewards, dialogue, location
- Storyline quest chains with progress indicators
- Procedural daily and weekly quests
- Quest tracking with navigation waypoints
- Abandon and restart options for repeatable quests
