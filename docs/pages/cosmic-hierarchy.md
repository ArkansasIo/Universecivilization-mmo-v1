# Cosmic Hierarchy

**Route:** `/cosmic-hierarchy`

## Purpose
A visualization of the structural hierarchy of the universe — from cosmic web filaments down to individual planets — for educational and navigational reference.

## Key Features
- Hierarchical tree view: Universe → Galaxy → Sector → System → Planet
- Breadcrumb navigation through hierarchy levels
- Entity counts and statistics at each level
- Zoom transitions between hierarchy layers
- Search and jump-to-location across levels
- Lore tooltips describing each structural level
