# Megastructures

- **Route:** `/megastructures`
- **Purpose:** Construction and management of mega-scale structures.

## Key Features

- Category filtering: all, energy, military, research, economic, special
- Tier system: Legendary, Epic, Rare
- Structure statuses: operational, construction, upgrading, damaged, locked
- Module system with installation costs and effects
- Resource requirements include metal, crystal, deuterium, dark matter, and antimatter
- Unlock requirements system

## UI Sections

| Section | Description |
|---|---|
| **Category Filters** | Tabs to filter megastructures by type category. |
| **Construction Queue** | Active construction/upgrade queue with progress tracking. |
| **Megastructure Cards** | Cards for each megastructure (e.g., Dyson Sphere, Science Nexus, Mega Shipyard, Planetary Shield) showing name, type, tier, level, status, description, effects, and requirements. |
| **Detail Panel** | Expanded view of a selected megastructure showing: stats (power output, defense rating, research bonus, etc.), installed modules with individual costs, construction progress, and upgrade options. |
| **Action Modals** | Separate modals for construction, upgrade, repair, and module management. |
