# Diplomacy Map

**Route:** `/diplomacy-map`

## Purpose
A visual map of inter-faction relationships — alliances, wars, trade pacts, and espionage links — shown as a force-directed graph.

## Key Features
- Interactive graph of factions with relationship edges
- Color-coded edges: green (ally), red (war), blue (trade), dashed (espionage)
- Click a node to view full diplomatic profile
- Timeline slider showing relationship changes over time
- Filter by relationship type or faction size
- Cluster detection showing diplomatic blocs
