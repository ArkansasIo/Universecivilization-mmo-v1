# Motherships

**Route:** `/motherships`

**Purpose:** Manage motherships — your capital ships with fleet-wide bonuses.

**Key Features:**
- Uses `useMothershipManager` for state and actions
- View and manage all owned motherships
- Spend resources on mothership upgrades

**UI Sections:**
- **Mothership Fleet** — list of owned motherships with name, class, level, and stats
- **Upgrades** — upgrade tree per mothership showing available improvements, costs, and stat bonuses
