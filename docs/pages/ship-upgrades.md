# Ship Upgrades

**Route:** `/ship-upgrades`

**Purpose:** Upgrade individual ships to improve their combat performance.

**Key Features:**
- Uses `useShipUpgrades` for upgrade state and actions
- Branching upgrade paths with tiered improvements
- Stat improvement previews before committing resources

**UI Sections:**
- **Upgrade Paths** — tree or branch view showing available upgrades per ship class
- **Cost Panel** — resource and credit costs for each upgrade tier
- **Stat Improvements** — before/after comparison of hull, shields, weapons, speed, and other stats
