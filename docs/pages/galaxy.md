# Galaxy

**Route:** `/galaxy`

## Purpose
The primary view of the galaxy — a 2D top-down map showing all star systems, sectors, and celestial bodies. Serves as the main navigation hub for exploring the universe.

## Key Features
- Rendered 2D map with star systems, nebulae, and sector boundaries
- Click-to-select systems for travel, scanning, or inspection
- Zoom and pan navigation
- Filter overlays (resource hotspots, faction control, threat levels)
- Quick-jump bookmarks to saved locations
- Real-time updates for fleet movements and system alerts
