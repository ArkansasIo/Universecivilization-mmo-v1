# NotFound

**Route:** `*` (catch-all)

**Purpose:** 404 page displayed for any route that does not match a registered path.

**Key Features:**
- Displays the attempted pathname via `useLocation`
- Large "404" background text with Orbitron font
- Compass icon and "Sector Not Found" heading
- Themed copy: "This sector of the galaxy has not yet been charted."
- Link back to home ("Return to Imperial Court")
- Ambient animated particle background
