# Game Test

**Route:** `/game-test`

## Purpose
A development/testing page for sandbox gameplay — allowing players to experiment with mechanics, spawn items, and test systems in an isolated environment.

## Key Features
- Sandbox mode with unlimited resources
- Item and ship spawning commands
- Combat test arena with configurable NPCs
- Economy simulation with adjustable parameters
- Reset and snapshot capabilities
- Debug overlays (coordinates, FPS, network latency)
