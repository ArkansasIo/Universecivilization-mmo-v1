# Espionage

**Route:** `/espionage`

## Purpose
Dedicated to covert operations — spying on other factions, stealing technology, sabotaging infrastructure, and counter-intelligence.

## Key Features
- Agent recruitment, training, and deployment
- Mission types: recon, theft, sabotage, assassination
- Success probability based on agent stats and target security
- Counter-espionage: detection networks and double agents
- Intel reports and intercepted communications
- Risk/reward balancing with faction reputation penalties
