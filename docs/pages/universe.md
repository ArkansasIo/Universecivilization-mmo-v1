# Universe

**Route:** `/universe`

## Purpose
An overview page displaying the entire universe structure — galaxies, clusters, and the cosmic web — giving players a sense of scale beyond a single galaxy.

## Key Features
- Top-level view of multiple galaxies and their interconnections
- Statistics: total systems, sectors, empires, population
- Cosmic web visualization showing galactic filaments
- Search across all galaxies for specific systems or players
- Telemetry feed of major universe-wide events
