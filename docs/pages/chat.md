# Chat

**Route:** `/chat`

## Purpose
Real-time messaging system with global, faction, system-local, and private channels for live player communication.

## Key Features
- Channel tabs: global, faction, system, alliance, private
- Emotes, formatting, and embedded item links
- Chat commands and slash-command interface
- Message history scrolling and search
- Mute, ignore, and moderation tools
- Notification settings per channel (ping, highlight keywords)
