# Crafting Skill Trees

- **Route:** `/crafting-skill-trees`
- **Purpose:** Crafting skill trees hub with five specializations.

## Key Features

- Skill points earned through crafting activities
- Five sub-trees:
  - **Weaponsmithing** — melee and ranged weapon crafting
  - **Armorsmithing** — armor and shield crafting
  - **Engineering** — ship modules and mechanical items
  - **Alchemy** — potions, consumables, and chemical crafting
  - **Nanotechnology** — advanced nano-augmentations
- Tree-switching with persistent progress per tree
- Passive crafting bonuses unlocked per branch
