# Resource Trading

**Route:** `/resource-trading`

## Purpose
A dedicated exchange for raw and refined resources — ores, gases, exotic matter, and fuels — with dynamic pricing based on supply and demand.

## Key Features
- Real-time price ticker per resource type
- Supply/demand ratios driving price fluctuation
- Regional price variation across sectors
- Limit and market orders for resource lots
- Resource conversion/refinement calculator
- Trade history and price alerts
