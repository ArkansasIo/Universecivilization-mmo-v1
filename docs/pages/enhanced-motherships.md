# Enhanced Motherships

**Route:** `/enhanced-motherships`

**Purpose:** Enhanced management interface for motherships with full stat visibility.

**Key Features:**
- Comprehensive stats display for all mothership attributes
- Upgrade management with detailed improvement previews
- Fleet bonus breakdown per mothership

**UI Sections:**
- **Mothership Overview** — expanded cards showing class, tier, level, and all stats
- **Stat Details** — hull, shields, power, crew, weapon mounts, and special abilities
- **Fleet Bonuses** — list of passive bonuses this mothership grants when assigned to a fleet
