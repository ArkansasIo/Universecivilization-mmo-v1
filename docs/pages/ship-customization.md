# Ship Customization

**Route:** `/ship-customization`

**Purpose:** Configure weapon and module loadouts on individual ships.

**Key Features:**
- Slot-based equipment system (primary, secondary, special weapons)
- Drag-and-drop or click-to-equip interface
- Real-time stat updates as loadout changes

**UI Sections:**
- **Ship Selector** — choose which ship to customize
- **Weapon Slots** — primary, secondary, and special weapon slots showing equipped items and empty slots
- **Inventory Panel** — available weapons and modules to equip, filtered by slot type
- **Stats Preview** — updated ship stats reflecting the current loadout
