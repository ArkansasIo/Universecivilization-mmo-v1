# Changelog

**Route:** `/changelog`

## Purpose
A public log of game updates, patches, hotfixes, and feature releases — keeping players informed of changes.

## Key Features
- Chronological list of patches with version numbers
- Categorization: new features, balance changes, bug fixes, performance
- Per-patch detail expansion for full notes
- Community-requested feature tags
- Developer commentary on major changes
- RSS/notification feed subscription
