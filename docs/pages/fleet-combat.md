# Fleet Combat

**Route:** `/fleet-combat`

**Purpose:** Simulate and auto-resolve fleet vs. fleet space combat.

**Key Features:**
- Configure attacker and defender fleets
- Auto-resolve combat with calculated outcomes
- View detailed battle logs and damage breakdowns

**UI Sections:**
- **Fleet Configuration** — side-by-side panels for attacker and defender fleet composition
- **Auto-Resolve Button** — trigger combat simulation
- **Results Panel** — outcome summary, surviving ships, resources lost
- **Battle Log** — turn-by-turn or phase-by-phase combat report
