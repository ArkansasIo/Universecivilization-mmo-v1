# Sectors

**Route:** `/sectors`

## Purpose
A catalog and management page for all defined sectors — regions of space grouped by proximity, faction control, or resource classification.

## Key Features
- List and grid views of sectors with territory overlays
- Sector detail panels: controlled by, resource abundance, stargate links
- Filtering by faction, security level, resource type
- Sector claim/colonization interface
- Sector-wide modifier display (production bonuses, hazard penalties)
- Activity feed per sector (fleet movements, conflicts, discoveries)
