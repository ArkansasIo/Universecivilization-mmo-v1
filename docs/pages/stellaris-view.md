# Stellaris View

**Route:** `/stellaris-view`

## Purpose
A strategic, top-down empire management overlay inspired by grand-strategy games — offering a data-rich view of territory, fleets, and development.

## Key Features
- Empire-wide territory map with sector borders
- Fleet deployment overview and movement orders
- Construction queue and development tracking
- Resource flow diagram across systems
- Diplomatic stance and relations summary
- Empire statistics panel (population, economy, military)
