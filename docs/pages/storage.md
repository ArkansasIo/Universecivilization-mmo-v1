# Storage

- **Route:** `/storage`
- **Purpose:** Overview and management of resource storage capacity across the empire.

## Key Features

- Planet selector to switch between colonies
- Category filters: all, metal, crystal, deuterium, universal, special
- Expandable storage module detail cards
- Shows current storage levels against max capacity with progress bars

## UI Sections

| Section | Description |
|---|---|
| **Storage Overview Cards** | High-level stats per resource type (Metal, Crystal, Deuterium, Energy) showing current capacity, max capacity, and utilization percentage. |
| **Storage Module Grid** | Card grid of storage structures (e.g., Metal Storage Silo, Crystal Vault, Deuterium Tank Complex, Universal Depot). Each card shows type, class (basic/advanced/elite/quantum), level, capacity used/max, efficiency %, protection %, installed modules, and upgrade costs. |
| **Module Detail Panel** | Expanded view of a selected storage structure showing installed modules (e.g., Compression Module, Security System, Auto-Sorting) with individual status. |
