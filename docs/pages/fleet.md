# Fleet

**Route:** `/fleet`

**Purpose:** Deploy ships on missions and manage your active fleet.

**Key Features:**
- Uses `useFleetManager` and `useResources` hooks
- Deploy and recall ships to/from missions
- Auto-fill fleet slots with optimal ships

**UI Sections:**
- **Fleet List** — table of all owned ships with status, stats, and current assignment
- **Mission Dispatch** — panel to select a mission and assign ships
- **Fleet Slots** — visual grid showing available fleet positions and occupied slots
