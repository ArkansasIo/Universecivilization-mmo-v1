# Stargate Network

**Route:** `/stargate-network`

## Purpose
Shows the network of stargates connecting systems and sectors, enabling fast travel across the galaxy.

## Key Features
- Graph visualization of gate nodes and links
- Route planning with travel time and toll costs
- Gate status indicators (active, charging, offline, contested)
- Owner/guild affiliation for private gates
- History of gate usage and events (attacks, malfunctions)
- Gate construction and upgrade interface
