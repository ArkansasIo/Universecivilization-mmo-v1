# Starships

**Route:** `/starships`

**Purpose:** Browse and compare all ship classes in the game.

**Key Features:**
- Complete catalog of ship classes with detailed stats
- Role-based filtering (attacker, support, scout, tank, etc.)
- Counter and synergy information between ship types

**UI Sections:**
- **Ship Class Browser** — scrollable list/grid of all ship classes with name, tier, and role badges
- **Stats Panel** — expanded view of selected ship showing hull, shields, speed, weapons, cargo capacity
- **Roles & Counters** — labels and tooltips indicating effective against / weak against relationships
