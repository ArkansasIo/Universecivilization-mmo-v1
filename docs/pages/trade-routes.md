# Trade Routes

**Route:** `/trade-routes`

## Purpose
Visualizes and manages trade routes — automated or player-run shipping lanes that generate profit by moving goods between systems.

## Key Features
- Map overlay of active and historical trade routes
- Route profitability calculator (distance, volume, tariffs)
- Route creation: origin, destination, cargo type, schedule
- Risk assessment (piracy chance, faction hostility)
- Fleet assignment to protect or run routes
- Route performance analytics (credits/hour, losses)
