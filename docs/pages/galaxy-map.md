# Galaxy Map

**Route:** `/galaxy-map`

## Purpose
An interactive, zoomable map view focused on spatial navigation and strategic planning across the entire known galaxy.

## Key Features
- Pan-and-zoom with infinite canvas mechanics
- Layer toggles: territories, trade routes, stargates, hazards
- Distance measurement tool between systems
- Route planning with waypoint placement
- Export/share screenshot of map region
- Animated travel preview for plotted routes
