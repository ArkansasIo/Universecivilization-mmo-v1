# Seed Discovery

**Route:** `/seed-discovery`

## Purpose
Tracks and manages procedurally generated discovery seeds — unique system/planet configurations that players can catalog, share, and revisit.

## Key Features
- Discovered seeds list with generation parameters
- Seed detail: system layout, resources, anomalies
- Share and bookmark seeds with other players
- Seed rarity classification (common, rare, legendary)
- Procedural generation preview for seed testing
- Seed comparison and filtering by attributes
