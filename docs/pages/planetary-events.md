# Planetary Events

**Route:** `/planetary-events`

## Purpose
Dynamic events that occur on or around specific planets — colony invasions, resource surges, environmental catastrophes, and discoveries.

## Key Features
- Active planetary events with location and countdown
- Event types: invasion, meteor storm, resource bloom, plague, discovery
- Player participation options (fight, evacuate, exploit, research)
- Per-planet event history and frequency stats
- Impact tracking: population, economy, environment
- Reward tiers based on contribution level
