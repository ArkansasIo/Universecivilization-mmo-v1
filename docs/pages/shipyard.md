# Shipyard

**Route:** `/shipyard`

**Purpose:** Construct new ships from available blueprints.

**Key Features:**
- Browse all buildable ship classes with their requirements
- Queue multiple ships for sequential construction
- Track build time and resource costs per ship

**UI Sections:**
- **Buildable Ships** — grid/cards of available ship classes with stats and costs
- **Construction Queue** — ordered list of ships currently being built with progress bars and time remaining
- **Build Time/Costs** — detailed breakdown of resource requirements (metal, crystal, credits) and construction duration per ship
