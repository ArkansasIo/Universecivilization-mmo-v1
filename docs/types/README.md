# Types

Shared TypeScript type definitions used across the project.

| File | Purpose |
|---|---|
| [gameTypes.md](gameTypes.md) | Core game entity interfaces: ships, buildings, technologies, resources, combat, systems |

## gameTypes.ts

Key types:
- `Ship` — Base ship stats
- `EnhancedShip` — Enhanced ship with nested hull/shields/power/crew/weapons
- `ResearchTech` — Research technology definition
- `Building` — Building definition
- `Resource` — Resource production/consumption
- `FleetMission` — Fleet movement orders
- `CombatResult` — Combat outcome data
- Systems, planets, colonies, and more
