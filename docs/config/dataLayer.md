# Config — Data Layer

**File:** `src/config/dataLayer.ts`

## Purpose
Data layer configuration — controls game data constants, generation parameters, and feature flags.

## Key Exports
- `DATA_LAYER_CONFIG` — Configuration object with `cosmicHierarchy` settings (seeds, universe count, galaxy count per universe, sector count per galaxy), `generation` settings (name generation options), and `features` (enableProcedural, enableDynamicEvents, etc.).
