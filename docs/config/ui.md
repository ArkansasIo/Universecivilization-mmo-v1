# Config — UI

**File:** `src/config/ui.ts`

## Purpose
UI configuration — theme colors, sizing, layout, and component settings.

## Key Exports
- `UI_CONFIG` — Object with `theme` (colors, fonts, spacing), `layout` (sidebar, panel, header sizes), `components` (table, button, modal, tooltip settings), `animations` (duration, easing), and `notifications` (toast duration, max visible).
