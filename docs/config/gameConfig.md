# Config — Game Config

**File:** `src/config/gameConfig.ts`

## Purpose
Central game configuration — master settings object that drives core game mechanics.

## Key Exports
- `GAME_CONFIG` — Object with `gameplay` (difficulty, speed, limits), `combat` (damage multipliers, accuracy, critical hit rates), `economy` (trade rates, tax rates, production multipliers), `exploration` (scan ranges, jump costs, anomaly chances), and `fleet` (maintenance, fuel consumption, command limits) settings.
