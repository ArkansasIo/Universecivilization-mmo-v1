# Config — Defaults

**File:** `src/config/defaults.ts`

## Purpose
Default game settings — starting values for player stats, resources, credits, ships, and game parameters.

## Key Exports
- `DEFAULT_PLAYER_STATS` — Initial stat values.
- `DEFAULT_RESOURCES` — Starting resources (credits, metal, crystal, etc.).
- `DEFAULT_SHIP` — Starting ship definition.
- `DEFAULT_GAME_CONFIG` — Master config with systems, combat, economy, exploration, and gameplay settings.
