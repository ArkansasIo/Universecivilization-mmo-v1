# Crafting Items

**File:** `src/data/craftingItems.ts`

## Purpose
Defines 90+ craftable items across categories: weapons, armor, modules, components, blueprints, technology, equipment, starships, consumables, drones, augmentations, and artifacts.

## Key Exports
- `craftingItems: CraftingItem[]` — Array of items with ids, names, type, class (basic–universal), category, tier, rarity, required level/skills, crafting time, materials, resource costs, stats, effects, and descriptions.
- `craftingCategories` — Array of all category labels for UI filtering.
