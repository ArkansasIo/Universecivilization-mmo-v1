# Enhanced Ships

**File:** `src/data/enhancedShips.ts`

## Purpose
Comprehensive enhanced ship system with detailed stats, game logic, and combat calculations.

## Key Exports
- `EnhancedShip` interface — Full ship definition with hull, shields, power, crew, weapons (primary/secondary/special), defense, mobility, capacity, systems, abilities, traits, upgrades, combat stats, maintenance, and requirements.
- `ShipGameLogic` class — Static methods for combat power calculation, damage calculation/application, shield regeneration, hull repair, fuel consumption, FTL checks, crew efficiency, maintenance costs, leveling, destruction checks, and retreat chance.
- `enhancedShips: EnhancedShip[]` — Example ship data (Eternal Titan Battleship).
