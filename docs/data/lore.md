# Lore

Galactic lore database with 100+ lore entries spanning history, factions, technology, and cosmic phenomena.

- `LORE_DATABASE: LoreEntry[]` — Array with id, title, category (history, faction, technology, cosmic, character, event, location, artifact), era, content, keyFigures, relatedEntries, significance, and discoveries.
