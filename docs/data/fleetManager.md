# Fleet Manager

## Purpose
Fleet management system for assembling, managing, and simulating fleet combat.

## Key Exports
- `FLEET_SIZE_LEVELS` — Fleet capacity by level (1–20), ranging from 1,000 to 2,000,000 capacity.
- `FleetManager` class — Methods for fleet creation, ship assignment/removal, fleet stats calculation, combat simulation, maintenance cost calculation, fleet splitting/merging, and fleet templates (scout, patrol, battle, invasion, carrier, stealth, support, expedition, defense, trade).
