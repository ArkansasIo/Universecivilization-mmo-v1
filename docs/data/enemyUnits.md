# Enemy Units

**File:** `src/data/enemyUnits.ts`

## Purpose
Enemy unit definitions for hostile NPC encounters, organized by faction.

## Key Exports
- `ENEMY_UNITS: EnemyUnit[]` — Array of enemy unit definitions with id, name, faction, type (infantry, vehicle, ship, building, drone, turret), class, tier, level, combat stats, abilities, loot table, AI behavior, and lore.
