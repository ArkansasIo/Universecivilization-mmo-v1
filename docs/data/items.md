# Items

**File:** `src/data/items.ts`

## Purpose
Comprehensive item database with weapons, armor, modules, consumables, and equipment — over 200+ item definitions.

## Key Exports
- `items: Item[]` — Array of item definitions with id, name, description, lore, type (Weapon, Armor, Module, Consumable, etc.), rank (E–SSS), rarity (Common–Universal), quality (Normal–Ascended), tier, level, maxLevel (50), baseStats, bonuses, cost, requirements, specialEffects, set bonuses, and upgrade paths.
- Items include weapons (laser cannons, torpedoes, railguns), armor plating, shield generators, engines, sensors, mining lasers, scanners, and special artifacts.
