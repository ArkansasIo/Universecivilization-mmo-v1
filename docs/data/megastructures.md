# Megastructures

**File:** `src/data/megastructures.ts`

## Purpose
Megastructure definitions — colossal space constructions with massive resource costs and game-changing effects.

## Key Exports
- `MEGASTRUCTURE_DATA: Megastructure[]` — Array of megastructure definitions with id, name, type (Dyson Sphere, Ring World, Alderson Disk, Matrioshka Brain, Orbital Elevator, Artificial Planet, Stellar Forge, Void Station, Citadel, Nexus Hub), tier (1–10), rank (SSS), rarity (Universal), constructionProgress, isComplete, effects (energy, research, production, military bonuses), requirements (technologies, resources, time), and upgrades.
