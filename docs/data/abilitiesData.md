# Abilities Data

**File:** `src/data/abilitiesData.ts`

## Purpose
Active and passive ability definitions for units, ships, and characters.

## Key Exports
- `ABILITY_DATA: AbilityData[]` — Array of ability definitions with id, name, type (active, passive), category (combat, support, utility, defensive, special), description, cooldown, energy cost, range, target type, effects, duration, tier, and unlock requirements.
