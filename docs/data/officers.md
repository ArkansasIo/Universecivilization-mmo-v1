# Officers

**File:** `src/data/officers.ts`

## Purpose
Officer/crew member definitions with classes, skills, and stat progression.

## Key Exports
- `officers: Officer[]` — Array of officer definitions with id, name, description, lore, class (Admiral, Commander, Scientist, Engineer, Diplomat, Tactician, etc.), rank (E–SSS), rarity, tier, level, maxLevel (50), baseStats (leadership, tactics, combat, intelligence, charisma, luck), skills with per-level effects, and assigned ship/fleet.
- Officers span from Novice Admiral to legendary figures (Admiral Zarathos, High Inquisitor Vex, Fleet Commander Aurelia, et al.).
